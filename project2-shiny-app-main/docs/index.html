TidyKit

A clean and interactive dashboard for data loading, preprocessing, feature engineering, and exploratory analysis. Built with R Shiny.

Try the App View Source Code

Core Features

📂

Data Loading

Import CSV, Excel, JSON, or RDS files. Two built-in sample datasets (iris & mtcars) are included for quick testing.

🧹

Data Cleaning

Handle duplicates, missing values, outliers, scaling, and categorical encoding through an interactive interface with real-time feedback.

🔧

Feature Engineering

Create transformed variables (log, square root, square, binning) with side-by-side visual comparison of original vs. transformed distributions.

📊

Exploratory Data Analysis

Interactive histograms, scatterplots, and boxplots. Dynamic summary statistics and correlation matrix. Optional linear regression overlay.

📖

User Guide

A built-in guided workflow helps users understand each step of the data analysis pipeline, from loading to visualization.

⚡

Responsive & Interactive

Polished UI with smooth navigation, dynamic controls, and instant visual feedback on every user action.

How to Run Locally

  1. Clone the repository: git clone https://github.com/Aurooora712/project2-shiny-app.git
  2. Open R or RStudio in the project directory
  3. Install dependencies: install.packages(c("shiny", "ggplot2", "readxl", "jsonlite", "DT", "shinyWidgets", "fontawesome"))
  4. Run the app: shiny::runApp()
